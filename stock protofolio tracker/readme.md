# 📈 Portfolio Tracker – Flask Web Application

A responsive and interactive **Stock Portfolio Tracker Web App** built using **Flask (Python), HTML5, and CSS3**.  
This project helps users calculate their total investment value based on stock symbols and quantities.

---

## 🌟 Overview

The Portfolio Tracker allows users to input multiple stock holdings and instantly calculates:

- Individual stock investment value
- Total portfolio value
- Clean tabular breakdown of holdings

This project demonstrates backend form handling using Flask and dynamic rendering using Jinja templates.

---

## 🚀 Features

✔ Add multiple stock entries  
✔ Real-time investment calculation  
✔ Displays stock-wise breakdown  
✔ Total portfolio valuation  
✔ Glassmorphism UI design  
✔ Fully responsive layout  
✔ Background image integration  

---

## 🧠 How It Works

1. User enters:
   - Stock symbol (e.g., AAPL)
   - Quantity of shares
2. Flask backend processes the form data
3. Each stock is matched with predefined prices
4. Investment is calculated:

\[
Investment = Stock Price \times Quantity
\]

5. Total value is summed and displayed in result page
