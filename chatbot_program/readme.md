# 🤖 SmartBot Pro – Flask Chatbot Web Application

A modern and interactive **Chatbot Web Application** built using **Flask (Python), HTML5, CSS3, and JavaScript**. This project enables users to communicate with a chatbot that responds to common queries such as greetings, date, time, jokes, and more through an attractive chat interface.

---

## 🌟 Overview

SmartBot Pro is a simple chatbot that demonstrates how a Flask backend communicates with a responsive frontend. User messages are processed by Python, and appropriate responses are displayed instantly without refreshing the webpage.

The project showcases backend routing, frontend interaction, and a modern user interface using Glassmorphism design.

## 🧠 How It Works

1. The user types a message in the chat input box.
2. JavaScript sends the message to the Flask backend using the Fetch API.
3. Flask receives the message through the **/get** route.
4. The **bot_reply()** function checks the user's input.
5. Based on the entered keywords, the chatbot generates an appropriate response.
6. The response is returned to the frontend in JSON format.
7. JavaScript displays the chatbot's reply in the chat window along with a typing animation.
